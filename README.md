# Titanic Survival by Age Analysis

This project looks at how age is related to survival in the Titanic dataset.

## What this project does

- loads the Titanic dataset
- checks how many age values are missing
- removes rows where age is missing
- creates a histogram of age by survival
- creates a KDE plot of age by survival

## Tools used

- Python
- seaborn
- matplotlib

## Output

This project shows:

- number of missing age values
- age distribution of passengers
- comparison of survivors and non-survivors by age

## What I learned

From this project, I learned how to:

- load a dataset using seaborn
- check missing values
- clean data before plotting
- compare groups using histogram and KDE plots
- understand age patterns in survival data

## How to run

1. Open the code in Google Colab or Jupyter Notebook
2. Run the code
3. View the charts

## Project goal

The goal of this project is to understand whether age had any visible relationship with survival in the Titanic dataset.
